# Trading Dashboard

An advanced trading analysis dashboard providing real-time technical analysis for NIFTY and SENSEX.

## Features

- **Real-time Charts**: High-performance candlestick and line charts.
- **AI Signal Detection**: Automated pattern recognition and signal generation.
- **Trendline Engine**: Advanced trendline detection and S/R level analysis.
- **Multi-Instrument**: Support for major Indian indices.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Frontend**: Vite, React, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Charts**: Lightweight-Charts
- **Backend Services**: Supabase (Edge Functions for Market Data)
